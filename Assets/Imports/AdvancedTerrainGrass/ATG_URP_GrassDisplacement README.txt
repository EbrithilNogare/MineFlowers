## ATG_URP_GrassDisplacement unitypackage

This package contains the ATG URP Grass Displacement scripts.

Make sure you have imported the "ATG_URP_7.1.7_Support" / "ATG_URP_7.2_Support" package first!

Importing the "ATG_URP_GrassDisplacement" package will add URP compatible scripts and a demo scene - all located in the folder "Lux SRP Grass Displacement" at the root of the assets folder.
In order to get grass displacement up and running please follow the instructions in the related "ATG URP Grass Displacement" pdf file.
